## 背景
我在学习大数据技术栈时遇到了很多问题，这些问题花费我大量的时间与头发。但我认为这并不是学习技术所必须经历的苦痛，因为其中大部分都是些配置与细节问题。本仓库的目标是为了后来者可以优化学习曲线，可以快速地对工具有一个整体的印象。如果对你的学习有帮助，求个Star !
